# Finance Tracker

Finance Tracker is a web-based personal finance management app deployed on **Vercel**, with **Firebase** for authentication and data storage. It helps users track transactions, manage categories, view analytics, and customize settings.

## Features
- Dashboard with financial overview
- Transaction & category management
- Analytics with charts/reports
- Secure authentication (Firebase)
- User settings & preferences

## Tech Stack
- Frontend: HTML, JavaScript, CSS  
- Backend/DB: Firebase (Auth + Firestore)  
- Deployment: Vercel  

## Getting Started
```bash
git clone https://github.com/vaillantm/finance-tracker.git
npm install
